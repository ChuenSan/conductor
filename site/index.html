// Features

多 Agent 并行的痛，都按住了

5 个 agent 同时跑的时候，真正的问题不是「怎么跑」，而是「我现在该看哪一个」。

🛰 Overview

Mission Control ⌘⇧M

全工作区 pane 实况卡片墙：思考计时、等回复、完成未读、屏幕预览。一眼定位该去哪，点卡片直达。

✋ Inbox

等你回复收件箱

agent 卡在权限确认？状态栏琥珀计数，弹层里 ⏎ / 1 / 2 / 3 / Esc 直接回，不用切 pane。

⏱ Live

思考活计时 + 完成闪绿

pane 头条实时走表「2:31」；跑完边框闪绿两下，后台完成亮绿点 + Dock 角标。

⛓ Queue

任务队列 ⌘⇧⏎

排队多条指令，agent 每次 Stop 自动接力下一条。下班挂机，早上收菜。

🔍 Review

二次意见

右键一下，让 Codex 审 Claude 刚写的方案（或反过来），分屏出严格的交叉质检报告。

🧬 Native

真终端，不是壳

ghostty 内核 + Metal 渲染 + 像素级对齐；工作区 / 分屏 / 会话续聊 / 误关恢复 / 中文 IME。随时跳进任何一个 pane 亲自接管。
