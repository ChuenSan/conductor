Terminal-first

Chrome is quiet and compact. Terminal surfaces own the page weight; no chat composer, no large decorative terminal cards.

Floating inspector

The sidebar is a lightweight card for workspace context and actions, collapsible without reparenting terminal surfaces.

Top workspace tabs

Workspaces are visible and renameable across the top, closer to a browser model than a dropdown picker.

Edge signals

Notifications and agent state are dots, badges, rings, and a right drawer, not persistent center-stage panels.
